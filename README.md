#  Portfolio

portfolio website for **Navis Joshva Donel J**, an Artificial Intelligence student and explorer.

## ✨ Features

- **Futuristic UI/UX**: Dark theme with glassmorphism and interactive elements.
- **Deeply Animated**: Powered by Framer Motion for smooth scroll reveals and transitions.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **High Performance**: Built with Vite and React for lightning-fast loading.

## 🚀 Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Visual Effects**: Three.js & React Three Fiber
- **Icons**: Lucide React

## 🛠️ Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone or download the project.
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create an optimized production build:
```bash
npm run build
```
The output will be in the `dist` directory.

## 🌐 Deployment

### Netlify

1. Connect your GitHub repository to Netlify.
2. Set the build command to `npm run build`.
3. Set the publish directory to `dist`.
4. Deploy!

### Vercel

1. Connect your GitHub repository to Vercel.
2. Vercel will automatically detect the Vite setup.
3. Click "Deploy".
